# Vora Language VS Code Extension

This extension provides rich language support for the Vora programming language in Visual Studio Code.

## Features

- Syntax highlighting
- Code completion
- Error diagnostics
- Code formatting
- Go to definition
- Integration with the Vora compiler

## Directory Structure

- `src/` - Extension source code
- `resources/` - Icons, themes, and other resources
