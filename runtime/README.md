# Vora Runtime

The Vora runtime provides the execution environment for Vora programs. It includes the standard library, memory management, and integration with the Majd Engine.

## Directory Structure

- `src/` - Runtime implementation
  - Standard library functions
  - Memory management
  - Majd Engine integration
  - Built-in types and operations
- `include/` - Runtime API headers
- `tests/` - Runtime unit tests
